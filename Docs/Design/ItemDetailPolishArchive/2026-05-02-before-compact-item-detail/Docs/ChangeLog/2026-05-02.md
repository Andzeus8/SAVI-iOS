# 2026-05-02 SAVI Engineering Ledger

## Summary

Started formal change tracking for the native SAVI iOS app and reconstructed the
Search/Home work completed during the day. No git commits existed for these
changes, so this entry is based on current source, file modification times,
simulator screenshots, and chat context.

## Search Changes

- Removed the visible example hint row that showed `Try: youtube`, `#recipe`,
  `PDFs`, and `screenshots`.
- Restored the Search bar placeholder to `Search links, files, notes, videos...`.
- Preserved the current Search structure at the time of logging:
  - Search header with save count.
  - Search bar.
  - File-type rail.
  - Facet controls for Type, Tags, Folders, Date, Source, and Has.
  - Timeline-style results.
- Added or preserved a visible `Clear all` path for active Search controls.
- Verified Search after simulator reboot on the main app bundle `com.savi.app`.

## Home And Result Row Changes

- Made folder tokens in Home/Search result rows quieter so saved item titles stay
  visually dominant.
- Folder tokens now use muted text, a subtle neutral capsule, and a small
  folder-color accent instead of a loud filled pill.
- Added Home bottom-tab tap behavior so tapping Home can request scroll-to-top.
- Restored a minimal thumbnail-corner type/source badge on compact saved-item
  rows so PDFs, links, videos, images, YouTube, TikTok, and other platforms are
  recognizable without adding noisy metadata pills.
- Rolled Search back from the newer large prompt/facet-bar experiment to the
  cleaner morning structure: `Find it fast`, compact search field, visible type
  rail, one `Refine` button for deeper filters, and a single clear-all path.
- Refined thumbnail type/source badges from colorful mini brand chips to a
  consistent white-on-dark outlined treatment so icons remain useful without
  distracting from saved-item titles.
- Archived the current SAVI `S` app icon set and generated the alternate
  `Share Pocket S` icon concept with a custom share-tray mark, refreshed 1024
  master, all required app icon sizes, and a comparison preview.
- Archived the `Share Pocket S` icon set and generated a cleaner `Clean Share S`
  concept with the background cards removed, a distinct cream share tray/arrow,
  and a separated lime `S` so the mark remains legible at home-screen and
  share-sheet sizes.
