# 2026-05-03 SAVI Change Log

## Summary

- Cleaned up Explore so it no longer brands the page as `SAVI Shuffle` or spends
  excessive vertical space explaining itself.
- Reworked the Folder editor into a more comfortable editor with a preview,
  name, collapsible Color/Image/Symbol/Privacy sections, and a visible
  `Find image` affordance.
- Fixed the Folder color-save bug where built-in Folder IDs were visually forced
  back to hard-coded colors after editing.

## Details

- Explore now uses a compact header:
  - `Explore`
  - `A fresh mix of your links, places, videos, and friends' saves.`
  - small status text
  - compact reset/shuffle buttons
- Removed duplicate `Today’s mix` and `SAVI Shuffle` header copy so the mosaic
  appears sooner.
- Kept Explore behavior intact: All/Mine/Friends scope, shuffle, reset history,
  friend actions, and the mosaic board.
- Folder editor now shows a live preview card at the top so color/icon/image
  changes are visible while editing.
- Color, Image, Symbol, and Privacy controls are grouped into collapsible
  sections so the editor no longer feels like one long wall of controls.
- `Find image` is now visible in the Folder editor even when the image-search
  proxy is not configured; tapping it explains that the image proxy is needed.
- Folder visual styling now respects saved custom colors. Default folders still
  keep their tuned default palette until the user changes the saved color.
